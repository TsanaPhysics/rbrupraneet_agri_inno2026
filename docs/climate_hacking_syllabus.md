# 12-Hour Workshop Syllabus: Climate Data Hacking with Python
## Project: RBRU-Praneet Digital Agri-Innovation Center

**Target Audience:** Secondary School Students (M.1-6 / Grade 7-12)  
**Duration:** 12 Hours (2 Days x 6 Hours)  
**Main Objective:** Learn Python programming by solving real-world climate challenges affecting agriculture in Chanthaburi.

---

## 📅 Day 1: Seeds of Code & Hunting for Data

### 🕘 Session 1: The Modern Farmer's Digital Toolkit (3 Hours)
*Focus: Environment Setup & Python Basics*
- **09:00 - 09:30:** Intro to "Agri-Inno" - Why does a durian farmer need a programmer?
- **09:30 - 10:30:** Setup Google Colab. Variables, Strings, and Integers.  
  *Activity:* Write a script to calculate the growth cycle of a fruit based on a planting date.
- **10:30 - 12:00:** Logic & Decisions (if/else).  
  *Activity:* Build a "Rain Warning" simulator—if the humidity is high, warn the farmer to check for fungi.

---

### 🕐 Session 2: Connecting the Farm to the Cloud (3 Hours)
*Focus: APIs & Real-time Sensors*
- **13:00 - 14:00:** Intro to APIs - How websites talk to each other.
- **14:00 - 15:30:** The `requests` library. Fetching climate data from the **TMD Open Data API** for Chanthaburi.
- **15:30 - 16:00:** Real-time Challenge: Display the current temperature at RBRU and compare it with the 5-year average.

---

## 📅 Day 2: The Data Detective & Smart Advisor

### 🕘 Session 3: Exploring the Pattern (3 Hours)
*Focus: Data Analysis with Pandas*
- **09:00 - 10:30:** Intro to `Pandas`. Reading CSV/Excel files like a pro.
- **10:30 - 12:00:** Historical Analysis - Analyzing 10 years of rainfall in Chanthaburi.  
  *Challenge:* Identify the years with the most "Unexpected Rain" during the flowering season.

---

### 🕐 Session 4: Designing the Agri-Inno Prototype (3 Hours)
*Focus: Visualization & Automation*
- **13:00 - 14:30:** Data Visualization with `Seaborn`. Plotting rainfall trends vs. fruit yield.  
  *Activity:* Create a "Climate Health Bar" for an orchard.
- **14:30 - 15:30:** The **Final Project: "Agri-Smart Advisor"**.  
  *Task:* Write a module that reads a 7-day forecast and sends a "Action Plan" (e.g., "High heat detected: Increase irrigation frequency by 20%").
- **15:30 - 16:00:** Demo & After Action Review (AAR).

---

## 📦 Software & Tools Required
1.  **Google Account:** To use Google Colab (no installation needed).
2.  **Browser:** Chrome or Firefox recommended.
3.  **TMD API Key:** (Staff will assist with registration before the workshop).

## 💡 Learning Outcomes
1.  Understand basic Python syntax and data structures.
2.  Be able to interact with REST APIs to fetch environmental data.
3.  Gain experience in data cleaning and visualization using industry-standard libraries.
4.  Develop a "Solution-Oriented" mindset for local agricultural issues.
