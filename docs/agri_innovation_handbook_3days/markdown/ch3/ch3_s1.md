## 3.1 ฟิสิกส์โซลาร์เซลล์ ความนำสารกึ่งตัวนำ และระบบพลังงานเก็บกักแบตเตอรี่
**(Photovoltaic Semiconductor Physics & Battery Systems)**

ในยุคของการเปลี่ยนผ่านด้านพลังงาน (Energy Transition) การทำความเข้าใจหลักการทางฟิสิกส์ที่อยู่เบื้องหลังการแปลงพลังงานแสงอาทิตย์ให้เป็นพลังงานไฟฟ้าอย่างมีประสิทธิภาพถือเป็นรากฐานสำคัญของการพัฒนาเกษตรอัจฉริยะ (Smart Farming) และระบบพลังงานนอกโครงข่าย (Off-grid System) บทนี้จะเจาะลึกถึงกลไกทางฟิสิกส์ของเซลล์แสงอาทิตย์ (Solar Cell) และหลักการทางเคมีไฟฟ้าของระบบจัดเก็บพลังงานแบตเตอรี่ (Battery Energy Storage System: BESS)

### 3.1.1 หลักการทางฟิสิกส์ของปรากฏการณ์โฟโตวอลเทอิก (Photovoltaic Effect)

ปรากฏการณ์โฟโตวอลเทอิก (Photovoltaic Effect) คือปรากฏการณ์ที่วัสดุบางชนิด โดยเฉพาะสารกึ่งตัวนำ (Semiconductor) สามารถสร้างกระแสไฟฟ้าได้โดยตรงเมื่อได้รับรังสีแม่เหล็กไฟฟ้า (Electromagnetic Radiation) ที่มีความถี่เหมาะสม ซึ่งในกรณีของพลังงานแสงอาทิตย์ (Solar Energy) คือการที่โฟตอน (Photon) จากแสงอาทิตย์มีพลังงานมากพอที่จะกระตุ้นให้เกิดการปลดปล่อยอิเล็กตรอน (Electron) ออกจากพันธะทางเคมี

#### 3.1.1.1 โครงสร้างสารกึ่งตัวนำแบบรอยต่อ p-n (p-n Junction)

หัวใจสำคัญของเซลล์แสงอาทิตย์ซิลิคอน (Silicon Solar Cell) คือการสร้างรอยต่อ p-n (p-n junction) ซึ่งเป็นโครงสร้างที่ประกอบด้วยสารกึ่งตัวนำชนิด p (p-type semiconductor) และสารกึ่งตัวนำชนิด n (n-type semiconductor)

1.  **สารกึ่งตัวนำชนิด n (n-type):** คือซิลิคอนที่ถูกเจือปน (Doping) ด้วยธาตุที่มีเวเลนซ์อิเล็กตรอน (Valence Electron) มากกว่าซิลิคอน เช่น ฟอสฟอรัส (Phosphorus) ทำให้มีอิเล็กตรอนอิสระ (Free Electrons) เป็นพาหะนำไฟฟ้าส่วนใหญ่ (Majority Carriers)
2.  **สารกึ่งตัวนำชนิด p (p-type):** คือซิลิคอนที่ถูกเจือปนด้วยธาตุที่มีเวเลนซ์อิเล็กตรอนน้อยกว่าซิลิคอน เช่น โบรอน (Boron) ทำให้เกิด "ช่องว่าง" (Holes) ซึ่งทำหน้าที่เป็นพาหะนำไฟฟ้าส่วนใหญ่

เมื่อนำสารกึ่งตัวนำทั้งสองชนิดมาสัมผัสกัน จะเกิดกระบวนการที่เรียกว่า **การเกิดเขตปลอดพาหะ (Depletion Region)** บริเวณรอยต่อ p-n นี้ จะเกิดการแพร่ (Diffusion) ของพาหะนำไฟฟ้าข้ามรอยต่อ โดยอิเล็กตรอนจากฝั่ง n จะแพร่ไปยังฝั่ง p และทำปฏิกิริยากับโฮล ทำให้เกิดไอออนที่ถูกตรึงไว้ (Fixed Ions) ซึ่งไอออนเหล่านี้จะสร้าง **สนามไฟฟ้าภายใน (Built-in Electric Field, $E_{bi}$)** ที่มีทิศทางจากฝั่ง n ไปยังฝั่ง p

#### 3.1.1.2 กลไกการสร้างกระแสไฟฟ้า (Photogeneration Mechanism)

เมื่อเซลล์แสงอาทิตย์ที่ประกอบด้วยรอยต่อ p-n ถูกฉายด้วยโฟตอน (Photon) จากแสงอาทิตย์:

1.  **การดูดซับโฟตอน (Photon Absorption):** โฟตอนที่มีพลังงาน ($E_{photon}$) มากกว่าหรือเท่ากับ **ช่องว่างพลังงาน (Band Gap Energy, $E_g$)** ของซิลิคอน (สำหรับซิลิคอน $E_g \approx 1.12$ eV) จะถูกดูดซับโดยสารกึ่งตัวนำ
2.  **การสร้างคู่พาหะ (Generation of Electron-Hole Pair):** พลังงานของโฟตอนจะถ่ายโอนไปยังอิเล็กตรอนในพันธะ ทำให้เกิดการกระตุ้นอิเล็กตรอนให้หลุดออกจากแถบวาเลนซ์ (Valence Band) ไปยังแถบนำไฟฟ้า (Conduction Band) ส่งผลให้เกิดคู่พาหะอิเล็กตรอน-โฮล (Electron-Hole Pair)
3.  **การแยกพาหะโดยสนามไฟฟ้า (Carrier Separation):** คู่พาหะที่เกิดขึ้นในเขตปลอดพาหะจะถูกแยกออกจากกันโดยสนามไฟฟ้าภายใน ($E_{bi}$) ที่มีอยู่แล้ว อิเล็กตรอน (Electron) จะถูกผลักไปทางฝั่ง n และโฮล (Hole) จะถูกผลักไปทางฝั่ง p
4.  **การไหลของกระแสไฟฟ้า (Current Flow):** การสะสมของประจุไฟฟ้าที่ขั้วทั้งสองด้าน (n-side เป็นลบ, p-side เป็นบวก) ทำให้เกิดความต่างศักย์ไฟฟ้า (Voltage) และเมื่อต่อวงจรภายนอก (External Circuit) อิเล็กตรอนที่สะสมตัวจะไหลผ่านวงจรภายนอกเพื่อกลับไปยังโฮลที่ปลายอีกด้านหนึ่ง ทำให้เกิดกระแสไฟฟ้ากระแสตรง (Direct Current: DC)

### 3.1.2 การคำนวณกำลังไฟฟ้าสูงสุดของแผงโซลาร์เซลล์ (Solar Panel Power Output Modeling)

การคำนวณกำลังไฟฟ้าที่แผงโซลาร์เซลล์สามารถผลิตได้ในสภาวะการทำงานจริงต้องพิจารณาปัจจัยทางกายภาพหลายประการ โดยเฉพาะอย่างยิ่งผลกระทบจากอุณหภูมิ (Temperature Effect)

กำลังไฟฟ้าที่แผงโซลาร์เซลล์ผลิตได้ ($P_{pv}$) สามารถประมาณได้จากสมการดังต่อไปนี้:

$$P_{pv} = \eta_{pv} \cdot A \cdot G_t \cdot [1 - \beta_{temp} \cdot (T_{cell} - 25)]$$

#### การอธิบายตัวแปรทางฟิสิกส์และวิศวกรรม:

| ตัวแปร | ชื่อเต็ม (English) | หน่วย SI | คำอธิบายทางวิชาการ |
| :--- | :--- | :--- | :--- |
| $P_{pv}$ | Power Output | Watt (W) | กำลังไฟฟ้าที่แผงโซลาร์เซลล์ผลิตได้ ณ สภาวะการทำงานปัจจุบัน |
| $\eta_{pv}$ | Photovoltaic Efficiency | Dimensionless | ประสิทธิภาพของแผงโซลาร์เซลล์ (ค่าที่ได้จากการทดสอบมาตรฐาน) |
| $A$ | Area | $\text{m}^2$ | พื้นที่ผิวรวมของแผงโซลาร์เซลล์ (Total Area) |
| $G_t$ | Irradiance | $\text{W}/\text{m}^2$ | ความหนาแน่นของรังสีดวงอาทิตย์ที่ตกกระทบ (Global Irradiance) |
| $T_{cell}$ | Cell Temperature | $\text{°C}$ | อุณหภูมิของเซลล์โซลาร์เซลล์ขณะทำงาน (Operating Cell Temperature) |
| $\beta_{temp}$ | Temperature Coefficient | $\text{W}/(\text{m}^2 \cdot \text{°C})$ | สัมประสิทธิ์อุณหภูมิของกำลังไฟฟ้า (Negative value) แสดงว่าเมื่ออุณหภูมิสูงขึ้น กำลังไฟฟ้าจะลดลง |
| $25$ | Reference Temperature | $\text{°C}$ | อุณหภูมิอ้างอิงมาตรฐาน (Standard Test Condition: STC) คือ $25 \text{ °C}$ |

**คำอธิบายเชิงลึก:**

1.  **ประสิทธิภาพ ($\eta_{pv}$):** คืออัตราส่วนของกำลังไฟฟ้าที่ผลิตได้ต่อกำลังไฟฟ้าที่ตกกระทบ ($\eta_{pv} = P_{out} / P_{in}$ )
2.  **ผลกระทบของอุณหภูมิ (Temperature Derating):** พลังงานไฟฟ้าที่ผลิตได้จะลดลงเมื่ออุณหภูมิของเซลล์สูงกว่าอุณหภูมิอ้างอิง ($25 \text{ °C}$) เนื่องจากเมื่ออุณหภูมิสูงขึ้น ความต่างศักย์ไฟฟ้า (Open-Circuit Voltage, $V_{oc}$) ของเซลล์จะลดลงตามหลักการทางฟิสิกส์ของสารกึ่งตัวนำ ทำให้กำลังไฟฟ้าโดยรวมลดลง การคำนวณส่วน $[1 - \beta_{temp} \cdot (T_{cell} - 25)]$ คือการปรับแก้กำลังไฟฟ้าตามอุณหภูมิที่วัดได้จริง

### 3.1.3 หลักการเคมีไฟฟ้าและระบบประจุพลังงานแบตเตอรี่ (Electrochemistry and Battery Charging)

ระบบพลังงานเก็บกักแบตเตอรี่ (BESS) ทำงานบนหลักการของปฏิกิริยารีดอกซ์ (Redox Reaction) ซึ่งเป็นการถ่ายโอนอิเล็กตรอนระหว่างสารเคมีที่ขั้วไฟฟ้า (Electrodes) ภายในแบตเตอรี่

#### 3.1.3.1 เคมีไฟฟ้าของแบตเตอรี่ลิเธียมฟอสเฟต (LiFePO$_4$)

แบตเตอรี่ลิเธียมฟอสเฟต ($\text{LiFePO}_4$) เป็นเคมีภัณฑ์ที่ได้รับความนิยมสูงในงานพลังงานขนาดใหญ่ (Grid-scale) และระบบเกษตรดิจิทัล เนื่องจากมีคุณสมบัติที่โดดเด่นคือ **ความเสถียรทางความร้อน (Thermal Stability)** และ **ความปลอดภัยสูง (High Safety)** เมื่อเทียบกับแบตเตอรี่ลิเธียมไอออนชนิดอื่น ๆ

**ปฏิกิริยาทางเคมีไฟฟ้า (Electrochemical Reactions):**

1.  **ที่ขั้วลบ (Anode, $\text{Li}_x\text{FePO}_4$):** ในระหว่างการคายประจุ (Discharging) ไอออนลิเธียม ($\text{Li}^+$) จะถูกปล่อยออกมาจากโครงสร้างของวัสดุที่ขั้วลบ
    $$\text{LiFePO}_4 \longrightarrow \text{FePO}_4 + \text{Li}^+ + e^-$$
2.  **ที่ขั้วบวก (Cathode, $\text{LiCoO}_2$ หรือวัสดุอื่น):** ไอออนลิเธียมที่ถูกปล่อยออกมาจะเดินทางผ่านอิเล็กโทรไลต์ (Electrolyte) ไปยังขั้วบวก และเกิดปฏิกิริยาการรวมตัว
    $$\text{FePO}_4 + \text{Li}^+ + e^- \longrightarrow \text{LiFePO}_4$$

**วงจรไฟฟ้าโดยรวม (Overall Cell Reaction):**
$$\text{LiFePO}_4 + \text{Electrolyte} \xrightarrow{\text{Discharge}} \text{FePO}_4 + \text{Li}^+ + e^-$$

**ข้อได้เปรียบของ $\text{LiFePO}_4$:** โครงสร้างทางเคมีของ $\text{FePO}_4$ มีความเสถียรสูงมาก ทำให้เกิดความร้อนสะสม (Heat Generation) น้อยกว่าเมื่อเกิดการลัดวงจรหรือการใช้งานเกินขีดจำกัด ซึ่งเป็นปัจจัยสำคัญอย่างยิ่งสำหรับระบบที่ติดตั้งภายนอกอาคาร (Outdoor Smart Farm)

### 3.1.4 พารามิเตอร์จำลองสถานะของแบตเตอรี่ (Battery State Parameters)

ในการควบคุมระบบพลังงานอย่างแม่นยำ จำเป็นต้องมีการติดตามพารามิเตอร์สำคัญ 2 ตัว คือ สถานะการชาร์จ และดัชนีสุขภาพ

#### 3.1.4.1 สถานะการชาร์จ (State of Charge: SOC)

**คำจำกัดความ:** SOC คือปริมาณพลังงานที่เหลืออยู่ในแบตเตอรี่เมื่อเทียบกับความจุสูงสุดที่กำหนดไว้ (Nominal Capacity) โดยทั่วไปมีหน่วยเป็นเปอร์เซ็นต์ ($\%$) หรือเทียบเท่ากับปริมาณประจุไฟฟ้า (Ampere-hour: $\text{Ah}$)

**การคำนวณ:**
$$\text{SOC} (\%) = \frac{\text{Current Capacity}}{\text{Nominal Capacity}} \times 100$$

การประมาณค่า SOC สามารถทำได้หลายวิธี เช่น วิธีการวัดแรงดันไฟฟ้า (Voltage Measurement) หรือวิธีทางออมเพอร์ (Coulomb Counting) ซึ่งเป็นวิธีที่แม่นยำที่สุดในการจำลองระบบ

#### 3.1.4.2 ดัชนีสุขภาพของแบตเตอรี่ (State of Health: SOH)

**คำจำกัดความ:** SOH คือการวัดความสามารถในการจ่ายพลังงานของแบตเตอรี่ในปัจจุบัน เทียบกับความสามารถเดิมที่เคยเป็นเมื่อแบตเตอรี่ใหม่ (New Capacity) SOH จะลดลงตามอายุการใช้งาน (Cycle Life) และการเสื่อมสภาพทางเคมี (Degradation)

**การคำนวณ:**
$$\text{SOH} (\%) = \frac{\text{Current Capacity}}{\text{Nominal Capacity (New)}} \times 100$$

การติดตาม SOH เป็นสิ่งสำคัญอย่างยิ่งในการวางแผนการบำรุงรักษา (Maintenance Planning) ของระบบพลังงานในฟาร์มขนาดใหญ่

### 3.1.5 การจำลองการทำงานของระบบพลังงาน (Python Simulation Example)

เพื่อสาธิตการคำนวณ SOC และ SOH ในระบบควบคุมพลังงานของฟาร์มอัจฉริยะ เราจะใช้ภาษา Python ในการจำลองการเปลี่ยนแปลงของพลังงานแบตเตอรี่ตามเวลาและรอบการใช้งาน

```python
# -*- coding: utf-8 -*-
"""
Module: Battery_System_Simulator
Purpose: จำลองการคำนวณ State of Charge (SOC) และ State of Health (SOH) 
         สำหรับระบบพลังงานแบตเตอรี่ LiFePO4 ในฟาร์มอัจฉริยะ
"""

import time

# =============================================================================
# 1. กำหนดค่าพารามิเตอร์เริ่มต้นของแบตเตอรี่ (Initial Parameters)
# =============================================================================

# ความจุที่ระบุเมื่อแบตเตอรี่ใหม่ (Nominal Capacity) หน่วยเป็น Ah
NOMINAL_CAPACITY_AH = 100.0 

# ความจุปัจจุบันหลังจากการเสื่อมสภาพ (Current Capacity after degradation)
current_capacity_ah = 95.0  # สมมติว่าเสื่อมสภาพไปแล้ว 5%

# อัตราการชาร์จ/ดิสชาร์จ (Current in Amperes)
# ค่าบวกคือการชาร์จ (Charging), ค่าลบคือการดิสชาร์จ (Discharging)
current_a = 10.0  # 10 Amperes

# แรงดันอ้างอิงของระบบ LiFePO4 (Nominal Voltage)
SYSTEM_VOLTAGE_V = 12.8

def estimate_soc(current_ah: float, max_capacity_ah: float) -> float:
    """คำนวณเปอร์เซ็นต์ State of Charge (SOC)"""
    return (current_ah / max_capacity_ah) * 100.0

def estimate_soh(max_capacity_ah: float, nominal_capacity_ah: float) -> float:
    """คำนวณเปอร์เซ็นต์ State of Health (SOH)"""
    return (max_capacity_ah / nominal_capacity_ah) * 100.0

# จำลองการทำงาน 5 ชั่วโมง
dt_hours = 1.0  # สเต็ปเวลาเป็นชั่วโมง
current_energy_ah = 50.0  # เริ่มต้นที่ประจุ 50 Ah (SOC 50% ของความจุปัจจุบัน)

print("--- เริ่มต้นการจำลองระบบแบตเตอรี่ Smart Farm ---")
print(f"แบตเตอรี่พิกัดความจุเริ่มต้น: {NOMINAL_CAPACITY_AH} Ah")
print(f"สุขภาพแบตเตอรี่ปัจจุบัน (SOH): {estimate_soh(current_capacity_ah, NOMINAL_CAPACITY_AH):.2f}%")
print(f"ความจุสูงสุดที่รับประจุได้ขณะนี้: {current_capacity_ah} Ah")
print(f"ประจุกระแสชาร์จเฉลี่ย: {current_a} A ที่แรงดัน {SYSTEM_VOLTAGE_V} V\n")

for hour in range(1, 6):
    # คำนวณปริมาณประจุไฟฟ้าที่เปลี่ยนแปลง (Ampere-hours change)
    delta_ah = current_a * dt_hours
    current_energy_ah = min(current_energy_ah + delta_ah, current_capacity_ah)
    
    soc = estimate_soc(current_energy_ah, current_capacity_ah)
    energy_wh = current_energy_ah * SYSTEM_VOLTAGE_V
    
    print(f"ชั่วโมงที่ {hour}: ประจุสะสม = {current_energy_ah:.2f} Ah | SOC = {soc:.2f}% | พลังงานสะสม = {energy_wh:.2f} Wh")
```
