## 3.3 การเขียนโค้ดอ่านเซนเซอร์ วิเคราะห์กราฟคาลิเบรตดินเหนียวตราด (Soil Moisture Sensor Calibration for Trat Clay)

# 3.3 การเขียนโค้ดอ่านเซนเซอร์ วิเคราะห์กราฟคาลิเบรตดินเหนียวตราด (Soil Moisture Sensor Calibration for Trat Clay)

ในฐานะนักวิชาการผู้เชี่ยวชาญด้านฟิสิกส์ประยุกต์และเกษตรดิจิทัล การทำความเข้าใจหลักการทางฟิสิกส์ที่อยู่เบื้องหลังการวัดค่าความชื้นในดิน (Soil Moisture Measurement) เป็นสิ่งสำคัญยิ่งก่อนการเขียนโค้ดใด ๆ การวัดค่าความชื้นในดินที่แม่นยำและเชื่อถือได้นั้น ไม่ได้อาศัยเพียงแค่การอ่านค่าดิจิทัล (Digital Reading) เท่านั้น แต่ต้องอาศัยการทำความเข้าใจปฏิสัมพันธ์ระหว่างสนามไฟฟ้า (Electric Field) กับคุณสมบัติทางไฟฟ้าของตัวกลาง (Dielectric Properties) อย่างลึกซึ้ง

บทนี้จะนำเสนอการวิเคราะห์เชิงทฤษฎีและปฏิบัติอย่างละเอียด ตั้งแต่หลักการทำงานของเซนเซอร์วัดความชื้นแบบเก็บประจุ (Capacitive Soil Moisture Sensor) ไปจนถึงการสร้างแบบจำลองทางคณิตศาสตร์ (Mathematical Model) สำหรับการสอบเทียบ (Calibration) เพื่อให้ได้ค่าความชื้นดินเชิงปริมาตร (Volumetric Water Content: VWC) ที่มีความแม่นยำสูงสำหรับสภาพดินเหนียวเฉพาะถิ่นอย่างดินเหนียวตราด

## 3.3.1 หลักการทำงานทางฟิสิกส์ของเซนเซอร์วัดความชื้นแบบเก็บประจุ (Capacitive Principle)

### 3.3.1.1 ความแตกต่างระหว่างเซนเซอร์แบบต้านทานและแบบเก็บประจุ

เซนเซอร์วัดความชื้นในดินในตลาดมีหลายประเภท แต่สำหรับงานวิจัยและงานภาคสนามที่ต้องการความทนทานสูงและค่าที่เสถียรในระยะยาว **เซนเซอร์แบบเก็บประจุ (Capacitive Sensor)** ถือเป็นทางเลือกที่เหนือกว่าเซนเซอร์แบบต้านทาน (Resistive Sensor) อย่างชัดเจน

1.  **เซนเซอร์แบบต้านทาน (Resistive Sensor):** เซนเซอร์เหล่านี้ทำงานโดยการวัดความต้านทานไฟฟ้า (Electrical Resistance) ระหว่างขั้วไฟฟ้าสองขั้ว เมื่อดินเปียก ความต้านทานจะลดลง เนื่องจากน้ำเป็นตัวนำไฟฟ้าที่ดีกว่าอากาศ อย่างไรก็ตาม ข้อเสียหลักคือการเกิดการกัดกร่อน (Corrosion) ของขั้วไฟฟ้าอย่างรวดเร็วเมื่อสัมผัสกับอิเล็กโทรไลต์ (Electrolyte) ในดินเป็นเวลานาน ทำให้ค่าที่วัดได้ผิดเพี้ยนและอายุการใช้งานสั้น
2.  **เซนเซอร์แบบเก็บประจุ (Capacitive Sensor):** เซนเซอร์เหล่านี้ไม่ได้วัดความต้านทาน แต่จะวัดค่าความจุไฟฟ้า (Capacitance, $C$) ซึ่งเป็นคุณสมบัติทางไฟฟ้าที่เกิดจากการกักเก็บประจุไฟฟ้า (Electric Charge) ระหว่างแผ่นตัวนำ (Conducting Plates) สองแผ่นที่ถูกแยกด้วยไดอิเล็กทริก (Dielectric) ซึ่งในกรณีนี้คือดินรอบ ๆ เซนเซอร์

### 3.3.1.2 กลไกทางฟิสิกส์ของการวัดค่าความจุไฟฟ้า

เซนเซอร์แบบเก็บประจุถูกออกแบบให้มีโครงสร้างเป็นตัวเก็บประจุ (Capacitor) โดยมีแผ่นตัวนำไฟฟ้า (Electrodes) สองแผ่นที่ขนานกัน และมีตัวกลาง (Medium) คือดินที่อยู่ระหว่างแผ่นตัวนำ

ค่าความจุไฟฟ้า ($C$) ของตัวเก็บประจุแบบขนาน (Parallel Plate Capacitor) ถูกกำหนดโดยสมการพื้นฐานทางไฟฟ้าสถิต (Electrostatics):

$$C = \frac{\varepsilon_r \varepsilon_0 A}{d}$$

เมื่อ:
*   $C$ คือ ค่าความจุไฟฟ้า (Capacitance) มีหน่วยเป็น ฟารัด (Farad, F)
*   $\varepsilon_0$ คือ ค่าสภาพยอมทางไฟฟ้าของสุญญากาศ (Permittivity of Free Space) มีค่าคงที่เท่ากับ $8.854 \times 10^{-12} \text{ F/m}$
*   $A$ คือ พื้นที่ผิวของแผ่นตัวนำ (Area) มีหน่วยเป็น ตารางเมตร ($\text{m}^2$)
*   $d$ คือ ระยะห่างระหว่างแผ่นตัวนำ (Distance) มีหน่วยเป็น เมตร ($\text{m}$)
*   $\varepsilon_r$ คือ ค่าคงที่ไดอิเล็กทริก (Relative Permittivity) หรือค่าคงที่ไดอิเล็กทริกสัมพัทธ์ (Dielectric Constant) ของตัวกลางที่อยู่ระหว่างแผ่นตัวนำ

**การวิเคราะห์ความสัมพันธ์กับความชื้น:**

ในบริบทของเซนเซอร์วัดความชื้นในดิน ตัวกลางที่อยู่ระหว่างแผ่นตัวนำคือดิน (Soil) ดังนั้น ค่า $\varepsilon_r$ จึงเป็นค่าคงที่ไดอิเล็กทริกของดิน ($\varepsilon_{r, \text{soil}}$) ซึ่งเป็นคุณสมบัติที่ขึ้นอยู่กับองค์ประกอบของดินและปริมาณน้ำในดิน

เมื่อปริมาณน้ำในดินเพิ่มขึ้น (ดินเปียกขึ้น) น้ำจะเข้าไปแทนที่อากาศในช่องว่างของดิน (Pore Space) เนื่องจากน้ำมีค่าคงที่ไดอิเล็กทริก ($\varepsilon_{r, \text{water}} \approx 80$) สูงกว่าอากาศ ($\varepsilon_{r, \text{air}} \approx 1$) อย่างมาก การเพิ่มขึ้นของน้ำจึงส่งผลให้ค่า $\varepsilon_{r, \text{soil}}$ เพิ่มขึ้นตามไปด้วย

ดังนั้น:
$$\text{Soil Moisture} \uparrow \implies \varepsilon_{r, \text{soil}} \uparrow \implies C \uparrow$$

เซนเซอร์จึงทำงานโดยการวัดค่าความจุไฟฟ้าที่แปรผันตามค่าคงที่ไดอิเล็กทริกของดิน ซึ่งเป็นตัวบ่งชี้ถึงปริมาณน้ำในดิน

## 3.3.2 ทฤษฎีการสอบเทียบเซนเซอร์ (Sensor Calibration Theory)

### 3.3.2.1 การแปลงค่าความจุไฟฟ้าเป็นความชื้นเชิงปริมาตร (VWC)

เซนเซอร์สมัยใหม่มักจะแปลงค่าความจุไฟฟ้า ($C$) ให้เป็นสัญญาณแรงดันไฟฟ้าอนาล็อก (Analog Voltage Output) ($V_{out}$) ซึ่งเป็นค่าที่ไมโครคอนโทรลเลอร์ (Microcontroller) เช่น ESP32 หรือ Arduino สามารถอ่านได้

เนื่องจากค่า $C$ มีความสัมพันธ์เชิงเส้น (Linear Relationship) กับค่า $\varepsilon_{r, \text{soil}}$ และค่า $V_{out}$ ก็มีความสัมพันธ์เชิงเส้นกับ $C$ ดังนั้น เราจึงสามารถสร้างสมการเชิงเส้น (Linear Equation) เพื่อแปลงค่าแรงดันไฟฟ้าที่อ่านได้ ($V_{raw}$) ให้เป็นเปอร์เซ็นต์ความชื้นดินเชิงปริมาตร (Volumetric Water Content: VWC %) ได้

$$VWC (\%) = f(V_{raw})$$

โดยทั่วไปแล้ว ความสัมพันธ์นี้สามารถประมาณได้ด้วยสมการเชิงเส้นรูปแบบ:

$$VWC = m \cdot ADC_{raw} + c$$

เมื่อ:
*   $VWC$ คือ ปริมาณน้ำในดินเชิงปริมาตร (Volumetric Water Content) มีหน่วยเป็นเปอร์เซ็นต์ ($\%$)
*   $ADC_{raw}$ คือ ค่าดิจิทัลดิบ (Raw Digital Value) ที่อ่านได้จากตัวแปลงอนาล็อกเป็นดิจิทัล (Analog-to-Digital Converter, ADC) ซึ่งมีช่วงตั้งแต่ 0 ถึง $2^{N}-1$ (เช่น 0 ถึง 4095 สำหรับ ADC 12-bit)
*   $m$ คือ ความชัน (Slope) ของกราฟการสอบเทียบ (Calibration Slope)
*   $c$ คือ จุดตัดแกน Y (Y-intercept) หรือค่าความชื้นเมื่อ $ADC_{raw} = 0$

### 3.3.2.2 การกำหนดจุดสอบเทียบ (Defining Calibration Points)

การหาค่า $m$ และ $c$ อย่างแม่นยำต้องอาศัยการวัดค่าในสภาวะที่ทราบค่าความชื้นอย่างแน่นอน ซึ่งเราเรียกว่า **จุดสอบเทียบ (Calibration Points)** โดยเฉพาะสำหรับดินเหนียวตราด (Trat Clay) ซึ่งมีคุณสมบัติทางความชื้นที่เฉพาะเจาะจง เราต้องกำหนดจุดสองจุดหลัก:

1.  **จุดความชื้นต่ำสุด (Dry Soil Point):** คือการวัดค่าเมื่อเซนเซอร์ถูกฝังในดินที่แห้งสนิท (Air-dry state) ณ จุดนี้ $VWC$ ควรมีค่าใกล้เคียง $0\%$
    *   ให้ค่า $ADC_{raw, \text{dry}}$ เป็นค่าที่อ่านได้
    *   กำหนดให้ $VWC_{\text{dry}} = 0\%$
2.  **จุดความชื้นสูงสุด (Wet Soil Point):** คือการวัดค่าเมื่อเซนเซอร์ถูกฝังในดินที่อิ่มตัวด้วยน้ำ (Saturated state) ณ จุดนี้ $VWC$ ควรมีค่าใกล้เคียง $100\%$
    *   ให้ค่า $ADC_{raw, \text{wet}}$ เป็นค่าที่อ่านได้
    *   กำหนดให้ $VWC_{\text{wet}} = 100\%$

### 3.3.2.3 การคำนวณค่าความชันและจุดตัดแกน Y

เราใช้ข้อมูลจากสองจุด $(ADC_{raw, \text{dry}}, 0\%)$ และ $(ADC_{raw, \text{wet}}, 100\%)$ เพื่อหาค่า $m$ และ $c$ โดยใช้สูตรความชัน (Slope-Intercept Form) ของสมการเส้นตรง:

**1. การหาค่าความชัน ($m$):**
$$m = \frac{VWC_{\text{wet}} - VWC_{\text{dry}}}{ADC_{raw, \text{wet}} - ADC_{raw, \text{dry}}} = \frac{100\% - 0\%}{ADC_{raw, \text{wet}} - ADC_{raw, \text{dry}}}$$

**2. การหาจุดตัดแกน Y ($c$):**
เนื่องจากเราทราบว่าเมื่อ $VWC = 0\%$ ค่า $ADC_{raw}$ คือ $ADC_{raw, \text{dry}}$ เราสามารถแทนค่านี้ในสมการ:
$$0 = m \cdot ADC_{raw, \text{dry}} + c$$
$$c = -m \cdot ADC_{raw, \text{dry}}$$

ดังนั้น สมการการสอบเทียบที่สมบูรณ์สำหรับดินเหนียวตราดคือ:

$$VWC (\%) = \left( \frac{100}{ADC_{raw, \text{wet}} - ADC_{raw, \text{dry}}} \right) \cdot ADC_{raw} - \left( \frac{100}{ADC_{raw, \text{wet}} - ADC_{raw, \text{dry}}} \right) \cdot ADC_{raw, \text{dry}}$$

## 3.3.3 การประยุกต์ใช้โค้ดและการกรองสัญญาณรบกวน (Implementation and Noise Filtering)

ในการอ่านค่าจากเซนเซอร์ในสภาพแวดล้อมจริง ค่าที่อ่านได้จาก ADC มักจะมีสัญญาณรบกวน (Noise) จากการเปลี่ยนแปลงอุณหภูมิ (Temperature Fluctuation) การรบกวนทางแม่เหล็กไฟฟ้า (Electromagnetic Interference) และการเปลี่ยนแปลงของสภาพแวดล้อมอื่น ๆ การใช้เทคนิคการกรองสัญญาณ (Signal Filtering) จึงเป็นสิ่งจำเป็นอย่างยิ่ง

### 3.3.3.1 การกรองค่าเฉลี่ยเคลื่อนที่ (Moving Average Filter)

Moving Average Filter (MAF) เป็นวิธีการกรองสัญญาณที่ง่ายและมีประสิทธิภาพในการลดสัญญาณรบกวนแบบสุ่ม (Random Noise) โดยการคำนวณค่าเฉลี่ยของชุดข้อมูลที่อ่านได้ในช่วงเวลาสั้น ๆ (Window Size)

ถ้า $S_i$ คือค่าที่อ่านได้ในรอบที่ $i$ และ $N$ คือขนาดของหน้าต่าง (Window Size) ค่าที่กรองแล้ว $\hat{S}_i$ จะคำนวณดังนี้:

$$\hat{S}_i = \frac{1}{N} \sum_{j=0}^{N-1} S_{i-j}$$

การใช้ MAF จะช่วยให้ค่า $ADC_{raw}$ ที่นำไปคำนวณ VWC มีความเสถียรและใกล้เคียงกับค่าจริงทางฟิสิกส์มากขึ้น

### 3.3.3.2 โค้ดตัวอย่าง C++ สำหรับ Arduino (ESP32)

โค้ดต่อไปนี้เป็นตัวอย่างการเขียนโปรแกรมในภาษา C++ สำหรับแพลตฟอร์ม Arduino (ที่ใช้ไมโครคอนโทรลเลอร์ เช่น ESP32) เพื่อดำเนินการอ่านค่าเซนเซอร์ การกรองสัญญาณ และการแปลงค่าเป็น VWC อย่างสมบูรณ์

 * @return ค่า ADC ดิบที่อ่านได้
 */
int read_sensor_value() {
  // อ่านค่าอนาล็อกจากขาที่กำหนด
  int raw_value = analogRead(SENSOR_PIN);
  
  // เก็บค่าที่อ่านได้ลงในบัฟเฟอร์เพื่อเตรียมใช้ในการกรอง
  raw_adc_buffer[buffer_index] = raw_value;
  
  // เลื่อนตำแหน่งตัวชี้บัฟเฟอร์ไปข้างหน้า (วนกลับไปที่ 0 เมื่อถึงขนาดสูงสุด)
  buffer_index = (buffer_index + 1) % FILTER_SIZE; 
  
  return raw_value;
}

/**
 * @brief ฟังก์ชันคำนวณค่าเฉลี่ยเคลื่อนที่ (Moving Average Filter)
 * @param latest_reading ค่าที่อ่านได้ล่าสุด
 * @return ค่า ADC ที่ผ่านการกรองแล้ว
 */
int calculate_moving_average(int latest_reading) {
  long sum = 0;
  for (int i = 0; i < FILTER_SIZE; i++) {
    sum += raw_adc_buffer[i];
  }
  return sum / FILTER_SIZE;
}

/**
 * @brief ฟังก์ชันคำนวณเปอร์เซ็นต์ VWC จากค่า ADC ที่ผ่านการกรองแล้ว
 * @param adc_value ค่า ADC ที่ผ่านการกรอง
 * @return เปอร์เซ็นต์ความชื้นเชิงปริมาตร (% VWC)
 */
float calculate_vwc(int adc_value) {
  // ตรวจสอบขอบเขตล่างและขอบเขตบนเพื่อป้องกันค่าล้น (Out of bounds constraint)
  if (adc_value <= ADC_RAW_DRY) return 0.0;
  if (adc_value >= ADC_RAW_WET) return 100.0;
  
  // การแปลงค่าความสัมพันธ์เชิงเส้น
  float vwc = ((float)(adc_value - ADC_RAW_DRY) / (float)(ADC_RAW_WET - ADC_RAW_DRY)) * 100.0;
  return vwc;
}
