## 1.1 ดุลภูมิอากาศและสรีรวิทยาของทุเรียนตะวันออก (Agro-Climatology and Eastern Durian Physiology)

# ตำราวิชาการ: ฟิสิกส์เกษตรและระบบนิเวศดิจิทัล
## บทที่ 1.1 ดุลภูมิอากาศและสรีรวิทยาของทุเรียนตะวันออก (Agro-Climatology and Eastern Durian Physiology)

---

### บทนำ: ความสำคัญของการทำความเข้าใจปฏิสัมพันธ์ระหว่างสิ่งแวดล้อมกับสรีรวิทยาพืช

การเพาะปลูกพืชเศรษฐกิจที่มีมูลค่าสูง เช่น ทุเรียน (Durio zibethinus) ไม่ได้ขึ้นอยู่กับการจัดการปัจจัยการผลิตเพียงด้านเดียว แต่ต้องอาศัยความเข้าใจอย่างลึกซึ้งในปฏิสัมพันธ์ระหว่างปัจจัยทางภูมิอากาศ (Climatic Factors) และกระบวนการทางสรีรวิทยา (Physiological Processes) ของพืช การที่พืชจะสามารถเจริญเติบโตได้อย่างสมบูรณ์ โดยเฉพาะอย่างยิ่งในระยะวิกฤต เช่น ระยะออกดอก (Flowering Stage) และระยะติดผล (Fruiting Stage) จำเป็นต้องมีการจัดการความเครียด (Stress Management) อย่างแม่นยำ

ในบริบทของทุเรียนตะวันออก (Eastern Durian) ซึ่งเป็นสายพันธุ์ที่มีความต้องการสภาพแวดล้อมที่เฉพาะเจาะจง การวิเคราะห์ทางฟิสิกส์เกษตร (Agro-Physics) จึงเข้ามามีบทบาทสำคัญในการสร้างแบบจำลองทางคณิตศาสตร์เพื่อทำนายสภาวะความเครียดที่พืชอาจเผชิญ เพื่อให้เกษตรกรสามารถปรับปรุงการจัดการน้ำและธาตุอาหารได้อย่างเหมาะสมที่สุด

---

### 1. การวิเคราะห์กายภาพและสรีรวิทยาการเจริญเติบโตของทุเรียนในระยะวิกฤต

#### 1.1 ลักษณะสรีรวิทยาของทุเรียนในระยะออกดอกและติดผล

ทุเรียนเป็นพืชที่จัดอยู่ในกลุ่มไม้ยืนต้นเขตร้อน (Tropical Evergreen Tree) การเจริญเติบโตของทุเรียนมีความซับซ้อนทางสรีรวิทยา โดยเฉพาะอย่างยิ่งในช่วงที่เกิดการสืบพันธุ์ (Reproductive Stage) ซึ่งเป็นระยะที่พืชมีความไวต่อสภาวะตึงเครียด (Stress Susceptibility) สูงมาก

**ระยะออกดอก (Flowering Stage):**
ในช่วงนี้ พืชต้องใช้พลังงานจำนวนมหาศาลในการสร้างโครงสร้างสืบพันธุ์ (Reproductive Structures) เช่น ดอกและรังไข่ การที่ต้นทุเรียนจะสามารถสร้างดอกที่สมบูรณ์และติดผลได้นั้น ต้องอาศัยการจัดการน้ำที่เหมาะสมอย่างยิ่ง หากเกิดภาวะขาดน้ำ (Drought Stress) ในช่วงนี้ จะส่งผลให้เกิดการยุบของดอก (Flower Abortion) หรือการติดผลที่ไม่สมบูรณ์ (Poor Fruit Set) เนื่องจากกระบวนการทางสรีรวิทยาที่เกี่ยวข้องกับการสังเคราะห์แสง (Photosynthesis) และการลำเลียงน้ำ (Transpiration) ถูกรบกวน

**ระยะติดผล (Fruiting Stage):**
เมื่อผลเริ่มขยายขนาด (Fruit Enlargement) ความต้องการน้ำของพืชจะเพิ่มขึ้นอย่างก้าวกระโดด (Exponential Increase in Water Demand) การขาดน้ำอย่างรุนแรงจะกระตุ้นให้เกิดการปิดปากใบ (Stomatal Closure) เพื่อลดการสูญเสียน้ำ (Water Loss) อย่างไรก็ตาม การปิดปากใบที่มากเกินไปจะส่งผลให้การแลกเปลี่ยนก๊าซ (Gas Exchange) ลดลง ทำให้การสังเคราะห์แสงลดลงตามไปด้วย ซึ่งส่งผลโดยตรงต่อการสะสมมวลชีวภาพ (Biomass Accumulation) และคุณภาพของผลผลิต

#### 1.2 กลไกการตอบสนองต่อความเครียดจากน้ำ (Water Stress Response Mechanism)

เมื่อพืชเผชิญกับภาวะขาดน้ำ ความเครียดจากน้ำ (Water Stress) จะส่งผลให้ศักย์น้ำในเนื้อเยื่อ (Water Potential, $\Psi$) ของพืชลดต่ำลง (become more negative) กลไกหลักที่พืชใช้ในการตอบสนองคือการลดอัตราการคายน้ำ (Transpiration Rate) ผ่านการควบคุมการเปิดปิดของปากใบ (Stomatal Control)

ความสัมพันธ์นี้สามารถอธิบายได้ด้วยสมการที่เกี่ยวข้องกับความดันไอน้ำ (Vapor Pressure) ซึ่งเป็นหัวใจสำคัญในการทำความเข้าใจการถ่ายเทน้ำจากใบสู่บรรยากาศ

### 2. แนวคิดทางฟิสิกส์เกษตร: ดัชนีความตึงเครียดจากน้ำและอุณหภูมิเรือนยอด

#### 2.1 ดัชนีความตึงเครียดจากน้ำของพืช (Crop Water Stress Index: CWSI)

**นิยามและหลักการ:**
ดัชนีความตึงเครียดจากน้ำของพืช (CWSI) เป็นตัวชี้วัดเชิงปริมาณ (Quantitative Index) ที่ใช้ประเมินระดับความเครียดจากน้ำที่พืชกำลังเผชิญอยู่ โดยอาศัยการเปรียบเทียบระหว่างสภาวะความชื้นที่พืชต้องการ (Potential Stress) กับสภาวะความชื้นที่พืชได้รับจริง (Actual Stress)

**หลักการทางฟิสิกส์:**
CWSI มักคำนวณโดยใช้ข้อมูลจากความชื้นในดิน (Soil Moisture) และข้อมูลสภาพอากาศ (Meteorological Data) โดยเฉพาะอย่างยิ่งการคำนึงถึงความแตกต่างระหว่างความชื้นที่ดินที่เหมาะสม (Optimal Soil Moisture) กับความชื้นที่ดินปัจจุบัน (Current Soil Moisture)

โดยทั่วไปแล้ว ค่า CWSI จะมีค่ามากกว่า 1.0 เมื่อพืชอยู่ในภาวะเครียดจากน้ำปานกลางถึงรุนแรง และค่าเข้าใกล้ 0 เมื่อพืชอยู่ในสภาวะน้ำที่เหมาะสม (Optimal Water Status)

**การประยุกต์ใช้:**
การติดตามค่า CWSI อย่างต่อเนื่องช่วยให้ผู้ปฏิบัติงานสามารถวางแผนการให้น้ำ (Irrigation Scheduling) ได้อย่างแม่นยำ แทนที่จะใช้วิธีการให้น้ำตามตารางเวลา (Time-based Irrigation) ซึ่งอาจนำไปสู่การสูญเสียทรัพยากรน้ำอย่างไม่จำเป็น

#### 2.2 อุณหภูมิเรือนยอด (Canopy Temperature)

**แนวคิด:**
อุณหภูมิเรือนยอด (Canopy Temperature, $T_c$) คืออุณหภูมิเฉลี่ยของกลุ่มใบและกิ่งก้านของพืชในพื้นที่จำกัด (Micro-environment) ซึ่งแตกต่างจากอุณหภูมิอากาศรอบข้าง (Air Temperature, $T_a$)

**ความสำคัญทางสรีรวิทยา:**
ในสภาวะที่พืชมีการคายน้ำสูง (High Transpiration) พลังงานความร้อนส่วนหนึ่งจะถูกนำไปใช้ในการเปลี่ยนสถานะของน้ำ (Latent Heat of Vaporization) ซึ่งมีผลให้ $T_c$ มีค่าต่ำกว่า $T_a$ (Cooling Effect)

เมื่อพืชประสบภาวะขาดน้ำ (Water Stress) การคายน้ำจะลดลงอย่างมาก (Stomatal Closure) ส่งผลให้กลไกการระบายความร้อนนี้ถูกปิดตัวลง ทำให้ $T_c$ มีแนวโน้มที่จะสูงขึ้นและเข้าใกล้ $T_a$ มากขึ้น การวัดความแตกต่างของอุณหภูมินี้จึงเป็นตัวบ่งชี้ทางอ้อม (Indirect Indicator) ที่มีประสิทธิภาพว่าพืชกำลังประสบกับภาวะตึงเครียดจากน้ำหรือไม่

### 3. การวิเคราะห์สมการดุลพลังงานของเรือนยอดพืช (Canopy Energy Balance Equation)

การทำความเข้าใจว่าพลังงานความร้อนและรังสีถูกถ่ายเทอย่างไรในระบบเรือนยอดพืช (Canopy System) เป็นหัวใจสำคัญของฟิสิกส์เกษตร สมการที่ใช้ในการวิเคราะห์นี้คือสมการดุลพลังงาน (Energy Balance Equation)

**สมการหลัก:**
พลังงานสุทธิที่ได้รับโดยเรือนยอดพืช ($R_n$) จะต้องถูกถ่ายโอนออกไปในรูปแบบต่างๆ ตามหลักการอนุรักษ์พลังงาน (Law of Conservation of Energy)

$$R_n = G + H + \lambda ET_o$$

#### 3.1 คำอธิบายตัวแปรทางฟิสิกส์และนิยามเชิงลึก

| ตัวแปร | ชื่อเต็ม (English Term) | หน่วย SI | คำอธิบายเชิงฟิสิกส์ |
| :--- | :--- | :--- | :--- |
| $R_n$ | Net Radiation (รังสีสุทธิ) | $\text{W}/\text{m}^2$ | คือพลังงานรังสีสุทธิที่เข้าสู่เรือนยอดพืช ($R_{net} = R_{solar} - R_{reflected}$ โดย $R_{solar}$ คือรังสีดวงอาทิตย์ และ $R_{reflected}$ คือรังสีที่สะท้อนกลับ) |
| $G$ | Soil Heat Flux (ฟลักซ์ความร้อนในดิน) | $\text{W}/\text{m}^2$ | คือการถ่ายเทความร้อนระหว่างพื้นผิว (ดิน) กับบรรยากาศผ่านการนำความร้อน (Conduction) และการพาความร้อน (Convection) |
| $H$ | Sensible Heat Flux (ฟลักซ์ความร้อนสัมผัส) | $\text{W}/\text{m}^2$ | คือการถ่ายเทความร้อนที่เกี่ยวข้องกับอุณหภูมิ (Temperature Gradient) ระหว่างเรือนยอดกับอากาศโดยรอบ (Convection) |
| $\lambda ET_o$ | Latent Heat Flux (ฟลักซ์ความร้อนแฝง) | $\text{W}/\text{m}^2$ | คือพลังงานความร้อนที่ใช้ในการเปลี่ยนสถานะของน้ำจากของเหลวเป็นไอ (Phase Change) ผ่านกระบวนการคายระเหยน้ำ (Evapotranspiration) |

**การวิเคราะห์เชิงลึกของแต่ละเทอม:**

1.  **$R_n$ (Net Radiation):**
    $R_n$ คือพลังงานทั้งหมดที่ขับเคลื่อนกระบวนการทางชีวภาพ (Photosynthesis) และการระเหยน้ำ พลังงานนี้ประกอบด้วยรังสีคลื่นสั้น (Shortwave Radiation) และรังสีคลื่นยาว (Longwave Radiation) การคำนวณที่แม่นยำต้องพิจารณาการดูดซับ (Absorption) และการสะท้อน (Reflection) ขององค์ประกอบต่างๆ ในเรือนยอด
2.  **$G$ (Soil Heat Flux):**
    $G$ สะท้อนถึงความร้อนที่สะสมอยู่ในชั้นดิน การเปลี่ยนแปลงของ $G$ บ่งชี้ถึงความสามารถในการกักเก็บความร้อนของระบบรากและดิน ซึ่งมีผลต่อการควบคุมอุณหภูมิของรากพืช
3.  **$H$ (Sensible Heat Flux):**
    $H$ เป็นตัวบ่งชี้ถึงความแตกต่างของอุณหภูมิระหว่างเรือนยอดกับอากาศ หาก $H$ มีค่าสูง หมายความว่ามีการถ่ายเทความร้อนสูง ซึ่งมักเกิดขึ้นเมื่อความแตกต่างของอุณหภูมิมีมาก
4.  **$\lambda ET_o$ (Latent Heat Flux):**
    $\lambda$ คือค่าความร้อนแฝงของการกลายเป็นไอ (Latent Heat of Vaporization) ซึ่งมีค่าคงที่ทางกายภาพ (ประมาณ $2.45 \text{ kHzJ}/\text{kg}$ ที่อุณหภูมิมาตรฐาน) $ET_o$ คืออัตราการคายระเหยน้ำอ้างอิง (Reference Evapotranspiration) ซึ่งเป็นตัวกำหนดปริมาณน้ำที่ระบบต้องสูญเสียไปในรูปของไอน้ำ

**การนำไปใช้ในการจัดการน้ำ:**
สมการนี้ช่วยให้เราเข้าใจว่า หากเราต้องการให้พืชมีการคายน้ำสูง (เพื่อระบายความร้อนและดึงสารอาหาร) เราต้องมั่นใจว่ามีพลังงานรังสี ($R_n$) เพียงพอ และต้องมีการจัดการน้ำที่เพียงพอเพื่อรองรับการสูญเสียพลังงานในรูปของ $\lambda ET_o$

### 4. ความสัมพันธ์ระหว่างอุณหภูมิ ความชื้น และความดันไอน้ำ (Vapor Pressure Deficit: VPD)

#### 4.1 แนวคิดพื้นฐานของความดันไอน้ำ (Vapor Pressure)

**ความดันไอน้ำ (Vapor Pressure, $P_v$):**
คือความดันที่ไอน้ำในสถานะแก๊ส (Water Vapor) ออกฤทธิ์เหนือของเหลว (Liquid Water) ในระบบปิด (เช่น ในใบไม้ หรือในน้ำในดิน)

**ความชื้นสัมพัทธ์ (Relative Humidity, $RH$):**
คืออัตราส่วนระหว่างความดันไอน้ำจริง ($P_v$) กับความดันไอน้ำอิ่มตัว ($P_{sat}$) ที่ระดับอุณหภูมินั้นๆ

$$RH = \frac{P_v}{P_{sat}} \times 100\%$$

#### 4.2 ความแตกต่างของความดันไอน้ำ (Vapor Pressure Deficit: VPD)

**นิยาม:**
VPD คือความแตกต่างระหว่างความดันไอน้ำที่อากาศสามารถบรรจุได้สูงสุด (Saturation Vapor Pressure, $P_{sat}$) กับความดันไอน้ำจริง ($P_v$) ที่มีอยู่ในอากาศ ณ ขณะนั้น

$$\text{VPD} = P_{sat} - P_v$$

**ความสำคัญทางสรีรวิทยา:**
VPD เป็นตัวแปรทางสภาพอากาศที่สำคัญที่สุดตัวหนึ่งในการควบคุมอัตราการคายน้ำ (Transpiration Rate) ของพืช

1.  **VPD สูง (High VPD):** หมายความว่าอากาศแห้งมาก (Air is Dry) ความแตกต่างของความดันไอน้ำระหว่างภายในใบ (ซึ่งมี $P_v$ สูง) กับภายนอก (ซึ่งมี $P_v$ ต่ำ) มีมาก ทำให้แรงขับเคลื่อน (Driving Force) ในการคายน้ำสูงมาก พืชจึงต้องเปิดปากใบกว้างเพื่อระบายความร้อน แต่เสี่ยงต่อการสูญเสียน้ำอย่างรวดเร็ว
2.  **VPD ต่ำ (Low VPD):** หมายความว่าอากาศมีความชื้นสูง (Air is Humid) ความแตกต่างของความดันไอน้ำมีน้อย ทำให้การคายน้ำช้าลง ซึ่งเป็นสภาวะที่เหมาะสมต่อการเจริญเติบโตของทุเรียนในแง่ของการลดความเครียดจากน้ำ

**การจัดการเชิงปฏิบัติ:**
การเฝ้าระวังค่า VPD ช่วยให้ผู้ปฏิบัติงานสามารถคาดการณ์การสูญเสียน้ำของพืชล่วงหน้า หากค่า VPD สูงอย่างต่อเนื่อง ควรพิจารณาการให้น้ำแบบสม่ำเสมอเพื่อรักษาศักย์น้ำในดินให้สูงพอที่จะรองรับอัตราการคายน้ำที่สูงนั้นได้

### 5. ตัวอย่างการคำนวณและการจำลองทางคอมพิวเตอร์ (Computational Modeling Example)

เพื่อสาธิตการประยุกต์ใช้หลักการทางฟิสิกส์เกษตร เราจะเขียนโค้ด Python เพื่อคำนวณค่า VPD และประมาณการอัตราการคายน้ำ (Transpiration Rate) โดยใช้สมการที่อ้างอิงจากหลักการทางฟิสิกส์

**วัตถุประสงค์:** คำนวณ $P_{sat}$ และ VPD จากอุณหภูมิและความชื้นสัมพัทธ์ที่วัดได้

**สมการที่ใช้ (Clausius-Clapeyron Equation Approximation):**
เราจะใช้สมการที่ประมาณค่าความดันไอน้ำอิ่มตัว ($P_{sat}$) จากอุณหภูมิ ($T$) โดยใช้สูตรของ Magnus:
$$P_{sat} = 6.112 \cdot \exp\left(\frac{17.67 \cdot T}{237.3 + T}\right)$$
โดยที่ $P_{sat}$ มีหน่วยเป็น $\text{kPa}$ และ $T$ มีหน่วยเป็น ${}^\circ\text{C}$

**โค้ด Python สำหรับการคำนวณ VPD:**

```python
import numpy as np
import math

# ----------------------------------------------------------------------
# ฟังก์ชันคำนวณความดันไอน้ำอิ่มตัว (Saturation Vapor Pressure: P_sat)
# อ้างอิงจากสมการของ Magnus (Magnus' Formula)
# P_sat มีหน่วยเป็น kPa, T มีหน่วยเป็น Celsius
# ----------------------------------------------------------------------
def calculate_psat(temperature_celsius: float) -> float:
    """
    คำนวณความดันไอน้ำอิ่มตัว (P_sat) ที่อุณหภูมิที่กำหนด
    :param temperature_celsius: อุณหภูมิอากาศ (°C)
    :return: ความดันไอน้ำอิ่มตัว (kPa)
    """
    # ค่าคงที่ทางฟิสิกส์ (Constants)
    A = 17.67  # ค่าคงที่ A ในสมการ Magnus
    B = 237.3  # ค่าคงที่ B ในสมการ Magnus
    
    # การคำนวณ P_sat ตามสมการ: P_sat = 6.112 * exp((A * T) / (B + T))
    psat = 6.112 * np.exp((A * temperature_celsius) / (B + temperature_celsius))
    return psat

# ----------------------------------------------------------------------
# ฟังก์ชันคำนวณความแตกต่างของความดันไอน้ำ (Vapor Pressure Deficit: VPD)
# ----------------------------------------------------------------------
def calculate_vpd(temperature_celsius: float, relative_humidity_percent: float) -> float:
    """
    คำนวณค่า VPD (kPa)
    """
    psat = calculate_psat(temperature_celsius)
    pv = psat * (relative_humidity_percent / 100.0)
    vpd = psat - pv
    return vpd

# ตัวอย่างการใช้งาน
temp = 32.5  # อุณหภูมิเรือนยอด (องศาเซลเซียส)
rh = 65.0    # ความชื้นสัมพัทธ์ (%)
vpd_val = calculate_vpd(temp, rh)
print(f"ค่า VPD ที่คำนวณได้: {vpd_val:.4f} kPa")
```
